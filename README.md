# SQL Basics Practice

## 📌 Project Overview

This repository contains my SQL practice queries covering the fundamental concepts of SQL used in Data Analytics. The purpose of this project is to strengthen my understanding of querying relational databases by solving real-world business problems using SQL.

The queries are written in MySQL and cover filtering, sorting, aggregation, joins, conditional statements, date functions, and basic business analysis.

---

## 📂 Dataset

The project uses sample business datasets consisting of the following tables:

* **Sales** – Shipment details including amount, boxes sold, customers, salesperson, and product.
* **People** – Salesperson information and team details.
* **Products** – Product information used for sales analysis.

---

## 📚 SQL Concepts Covered

* SELECT Statement
* WHERE Clause
* ORDER BY
* AND / OR Conditions
* BETWEEN
* IN Operator
* LIKE Operator
* Column Aliases (AS)
* Calculated Columns
* CASE Statement
* Date Functions (`YEAR()`, `WEEKDAY()`)
* INNER JOIN
* GROUP BY
* Aggregate Functions (`SUM()`, `COUNT()`)
* LIMIT Clause

---

## 📊 Business Questions Solved

* Display all sales records.
* Retrieve selected columns from a table.
* Create calculated columns.
* Filter records using multiple conditions.
* Sort records in ascending and descending order.
* Categorize sales amounts using the CASE statement.
* Find shipments based on amount and box conditions.
* Count shipments made by each salesperson.
* Compare products based on total boxes sold.
* Analyze product performance during a specific time period.
